## How do I deploy the project?

1. Create .env file
2. Write in the console ```composer install```
3. Then ```npm install```
4. Then ```php artisan key:generate```
5. Then ```php artisan migrate --seed```
6. Then ```php artisan jwt:secret```
6. Then ```npm run dev```
7. Then ```php artisan serve```
